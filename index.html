Multiplying the Square of a Binomial
(a + b)2

The Formula to Multiply the binomials is:
(ax + b)2 =
(ax)2 + (2ab)x + b2

When entering the values for a2 and b2,

DO NOT enter any variables with the values. Only enter the coefficients of each of the terms.

Enter variable a

Enter variable b

This will FOIL the binomial for you so that you do not have to do it by hand.

Example: (5x + 4)2
a = 5, b = 4

Enter variable a

Enter variable b

Example: (2x + 3)2
a = 2, b = 3

Enter variable a

Enter variable b
